Unistroke Gesture Recognition

Unistroke-Gestures

Unistroke-gestures are a way to input by drawing various lines. It can be simple straight lines, curved or closed into shapes. This package contains tools for working with these gestures.

Description

This package will allow you to implement unusual and interesting gameplay elements through the use of unistroke gestures recognition.

Features

This package contains tools for working with unistroke-gestures:

  • A custom editor for creating and editing a unistroke-gesture patterns.
  • High performance recognized built with Unity Job System and Burst compiler.
  • No training required. Gestures patterns can be added even in run time.
  • Easy to customize and modify.
